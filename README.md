# LootLink Discord Raid Sync Extension

Prototype Chrome extension.

## Install

1. Open Chrome.
2. Go to `chrome://extensions`.
3. Turn on **Developer mode**.
4. Click **Load unpacked**.
5. Select this folder: `lootlink-discord-extension`.

## Use

1. Open Discord web: `https://discord.com`.
2. Go to a raid signup channel.
3. Click **LootLink Sync** in the bottom-right.
4. The extension scans the currently visible channel for Raid-Helper / SoftRes links.
5. Click **Copy JSON** to inspect the extracted data.

## Notes

This v1 scans visible rendered Discord content. It does not yet auto-sync to LootLink or scan every server/channel.
